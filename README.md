# 언팔로우

필요한것 : 관계만 제거하면 됨

view의 메인에 버튼 추가, 팔로우 추가하는 스크립트를 복붙하여 일부 수정하는것이 필요,

controllers/user에 팔로우 코드 기반으로 언팔로우 동작 코드 작성

router./user에 코드와 controllers의 unfollow관련 추가, 기존 코드랑 구성이 같으니 변수 두가지만 바꾸면 충분함
